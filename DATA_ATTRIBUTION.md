# Community data attribution

The Pal names, Paldeck identifiers, breeding power values, work-suitability values, and breeding result matrix in `public/data/` are adapted from the public Palworld 1.0 dataset maintained by [Kimpton's Palworld Breeding Guide](https://palworld.kimpton.io/).

- Game snapshot: Palworld 1.0
- Source updated: July 12, 2026
- Imported and cross-checked: July 14, 2026
- Records: 289 Pals plus 11 crossover creatures

The interface, copy, information architecture, filters, SEO pages, and visual system in this repository are original work for Palworld Field Guide. Palworld names and game concepts belong to their respective owners. This is an independent fan-made resource.
